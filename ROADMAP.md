# Roadmap

This document outlines the planned development direction for MCP Audit. For completed features, see the [Changelog](CHANGELOG.md).

## Current Status

**Version**: v0.3.x (Beta)
**Stage**: Public Beta - Stable for daily use

MCP Audit is currently in public beta with stable support for Claude Code, Codex CLI, and Gemini CLI. The core tracking and reporting functionality is production-ready.

## Upcoming (v0.4.0)

Target: Q1 2025

- [ ] **Multi-model session tracking** - Track sessions that switch between models (e.g., Opus → Sonnet → Haiku)
- [ ] **Enhanced CLI commands** - Additional report filters, date ranges, and export options
- [ ] **Improved TUI layouts** - Customizable dashboard panels and themes

## Future (v0.5.0+)

- [ ] **Ollama CLI support** - Time-based tracking for local models (no token costs)
- [ ] **Dynamic pricing via LiteLLM** - Auto-fetch current model pricing
- [ ] **Session comparison tools** - Compare efficiency across sessions
- [ ] **Cost budgeting** - Set alerts when approaching spending limits

## Long-Term Vision

- **Cross-platform unified tracking** - Single dashboard for all AI coding tools
- **Team/enterprise features** - Aggregate tracking across team members
- **Integration ecosystem** - Plugins for VS Code, JetBrains, and other IDEs
- **MCP server benchmarking** - Standardized efficiency metrics for MCP tool authors
