# Gemma Tokenizer License Information

token-audit uses the Gemma tokenizer to provide 100% accurate token estimation for Gemini CLI sessions.

## What This Means for Users

- **Optional download** — Only needed for Gemini CLI 100% accuracy
- **No HuggingFace signup required** — Downloads from GitHub Releases
- **Works offline after download** — Cached locally at `~/.cache/token-audit/`
- **Subject to Gemma Terms of Use** — When using Gemini CLI features

### Installation

```bash
# Download tokenizer (optional, for Gemini CLI)
token-audit tokenizer download

# Check status
token-audit tokenizer status
```

### Without Tokenizer

If you don't download the tokenizer:
- **Claude Code**: Works perfectly (has native token counts)
- **Codex CLI**: Works perfectly (uses tiktoken)
- **Gemini CLI**: Works with ~95% accuracy (tiktoken fallback)
